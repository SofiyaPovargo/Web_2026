Отдел информационных технологий предприятия

Single Page Application (SPA)

Добро пожаловать в информационное приложение по Отделу информационных технологий предприятия!

Здесь вы можете ознакомиться с общей информацией и перейти к просмотру структуры отдела.

Цель отдела

Обеспечение стабильной работы IT-инфраструктуры, разработка и поддержка информационных систем предприятия.

Основные задачи

Администрирование сетей, поддержка пользователей, обеспечение информационной безопасности, разработка ПО.

Структура

Включает руководителя, системных администраторов, разработчиков, специалистов по безопасности и техподдержке.

Структура Отдела информационных технологий

Должность Описание обязанностей
CIO / Директор по IT Разработка IT-стратегии, контроль инфраструктуры, обеспечение кибербезопасности, влияние IT на бизнес.
Системный администратор Обслуживание серверов и сетей, ежедневное управление инфраструктурой, устранение неполадок.
Специалист техподдержки Прием заявок пользователей, диагностика ПК и ПО, консультации по использованию техники.
Специалист по безопасности Мониторинг угроз, аудит безопасности, разработка политики защиты IT-систем.
Разработчик ПО Создание и кастомизация ПО, настройка систем управления данными для предприятия.
Администратор БД Управление хранением и извлечением данных, оптимизация баз данных.
